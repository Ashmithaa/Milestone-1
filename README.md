# Salary Trends Analysis Across Industries

An end-to-end **Data Analytics project** that analyzes global salary trends across industries, education levels, experience, gender, and geography using **Excel and MySQL**.

---

## Project Overview

In today’s data-driven world, understanding salary structures helps organizations design fair pay systems and helps professionals make better career decisions.  

This project analyzes a large global salary survey dataset to uncover meaningful compensation patterns and presents insights through an interactive dashboard.

---

## Project Objectives

- Analyze salary trends across industries and job roles  
- Study the impact of **education level** on salary  
- Compare salary and bonus distribution across **genders**  
- Examine how **years of experience** influence compensation  
- Identify **top-paying industries, roles, cities, and countries**  
- Build an **interactive Excel dashboard**  
- Demonstrate end-to-end analytics using **SQL + Excel**

---

## Dataset Overview

**Total Records:** 24,205 employees  

### Key Attributes

**Demographic**
- Age Range  
- Gender  

**Professional**
- Industry  
- Job Title  
- Job Title Clarification  

**Compensation**
- Annual Salary  
- Additional Monetary Compensation (Bonus/Commission)

**Education**
- Highest Education Level  

**Experience**
- Years of Professional Experience (Overall)  
- Years of Experience in Field  

**Geography**
- Country, State, City

## Data Cleaning & Preprocessing

- Handled missing values in salary and compensation columns  
- Standardized job titles, industries, and geographic names  
- Converted salary fields into numeric format  
- Treated outliers after validation  
- Prepared a structured dataset for MySQL integration  

---

## MySQL Database Integration

- Created database and table structure  
- Imported cleaned dataset into MySQL  
- Assigned proper data types and indexing  
- Used SQL for large-scale querying  

---

## SQL Analysis

Key queries performed:

- Average salary by industry and gender  
- Salary distribution by education level  
- Highest-paying job titles by country  
- Bonus percentage by gender  
- Compensation trends by experience  
- Salary by city and industry
